빠른 시간 기록 버튼

홈 화면에 '빠른 기록' 버튼 추가 (마지막 사용 프로젝트 기본값). 클릭 → 프로젝트 선택만 → 저장.

⏱️ 2시간 • 📊 일일 기록 20% 증가 예상

실행 취소 (Undo)

삭제한 기록을 3초 내에 복구 가능. 현재 soft-delete 구조 → UI만 추가.

⏱️ 1.5시간 • 🔄 실수 방지

주간 시간 자동 계산

/analytics에서 '이번 주 총 시간' 표시. getTotalHoursForWeek() 이미 구현됨.

⏱️ 1시간 • 📈 통계 대시보드 완성도 향상

CSV 내보내기

/logs에서 '다운로드' 버튼 → 날짜, 프로젝트, 시간, 메모 포함. 기초 구조 준비됨.

⏱️ 2시간 • 📥 데이터 이식성 제공

🚀 핵심 기능 개선 (1-2주)

청구서 생성 & PDF 다운로드

프로젝트별 청구서 자동 생성 + PDF 다운로드. API 스켈레톤 준비됨. 도구: pdfkit 또는 html2pdf.

⏱️ 5시간 • 💰 핵심 가치: 청구서 자동화

검색 고도화 (태그, 상태 필터)

프로젝트 태그별 필터링 (#web, #consulting). 현재 텍스트 검색만 지원 → 필터 UI 추가.

⏱️ 3시간 • 🔍 복잡한 프로젝트 관리 용이

자동 타이머 기능

'시작' 버튼 → 자동 시간 측정. 정지 시 기록 저장. 구현: React useTimer 훅 + 실시간 업데이트.

⏱️ 8시간 • ⏲️ 병렬 작업 시 편의성 향상

🔧 기술 개선 (중기)

  • ✓ 테스트 커버리지 (80%): Jest + Playwright E2E 테스트. 현재 파서만 50개 케이스.
  • ✓ 클라우드 동기화: Supabase (PostgreSQL) 연동. Drizzle ORM 덕분에 DB 마이그레이션 최소화 가능.
  • ✓ 성능 최적화: Lighthouse 90+ 목표. React Query 캐싱, 번들 크기 최적화.
